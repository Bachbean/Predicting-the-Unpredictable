# Bifurcations and Nonlinear Systems in a 1-D Context
Although chaotic systems typically only appear in high or infinite dimension systems, we can gain a lot of insight from observing 1-dimensional nonlinear systems and then applying similar techniques to more complex systems. 

Regarding 1-D nonlinear systems, one of the most useful things we can do during analysis is observe any qualitative information when looking at the graphs of the function. Typically, even for 1-D systems, derivatives and integrals of the system equation can quite complicated; however, using data from certain points, such as intersections or roots, can allow for good approximations or inferences. On the common velocity versus position graph for example, any intersection with the x-axis is what's known as a fixed point, and their properties are very important for understanding bifurcations.

Bifurcations are sudden changes within systems that result in a qualitative difference, and the point at which this happens is called the bifurcation point. The common example used to illustrate bifurcations is the function f(x) = x^2 + r, where r is a real parameter. As r changes from negative to 0 to positive, notice the number of roots the function has switches from 2 to 1 to 0, all between a single point. Intersections are just one example of qualitative measurements; often, stability is much more useful, which is the property of a point or system based on if it can still return to a certain equilibrium under a small displacement.
